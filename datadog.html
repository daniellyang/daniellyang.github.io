Designing a design system

Context

My time at Datadog was my first experience working in a standalone design systems team. It was a completely new and inspiring experience, so I sought to learn as much as I could. Some highlights from my internship include:

  • Leading the redesign efforts for a more designer-friendly internal design system documentation site that would bridge the communication gap between designers and engineers
  • Auditing the entire design system and documenting which components met https://www.w3.org/WAI/WCAG2AA-Conformance and recommendations to achieve platform compliance
  • Creating and dog-fooding a process for designers to sync with engineers and see how engineers actually use Datadog during critical outages in order to help the design org gain more user empathy
